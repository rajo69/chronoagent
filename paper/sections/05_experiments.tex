\section{Experiments}
\label{sec:experiments}

\subsection{Claims}
\label{sec:experiments:claims}

We evaluate five claims:
\begin{description}
  \item[C1] Behavioral signals shift measurably under memory poisoning.
  \item[C2] The BOCPD + Chronos ensemble flags shifts with
  $\awt{} \geq 0$ across runs.
  \item[C3] Health-weighted allocation improves efficiency under
  attack relative to a no-monitoring round-robin baseline.
  \item[C4] \sys{} achieves competitive AUROC against a reactive
  Sentinel-style baseline thanks to temporal context.
  \item[C5] The Chronos forecaster can be run inline per PR-handling
  step without stalling the control plane. C5 is a soft, qualitative
  claim in this draft; a quantitative wall-clock microbenchmark is
  deferred to task 11.4 and the reproducibility harness.
\end{description}

\subsection{Configurations}
\label{sec:experiments:configs}

Six YAML configs (see \texttt{configs/experiments/}) define the study:
\begin{enumerate}
  \item \textbf{\texttt{main\_experiment}} -- MINJA attack, full
  \sys{} stack.
  \item \textbf{\texttt{agentpoison\_experiment}} -- AgentPoison
  attack, full stack.
  \item \textbf{\texttt{baseline\_sentinel}} -- reactive per-query
  filter baseline.
  \item \textbf{\texttt{ablation\_no\_bocpd}} -- forecaster only.
  \item \textbf{\texttt{ablation\_no\_forecaster}} -- BOCPD only.
  \item \textbf{\texttt{ablation\_no\_health\_scores}} -- detectors
  on, allocator insensitive to health.
\end{enumerate}

A seventh config, \texttt{signal\_validation} (Phase 1), establishes
C1 independent of the runtime stack.

\subsection{Protocol}
\label{sec:experiments:protocol}

Each configuration is executed via
\begin{verbatim}
chronoagent run-experiment phase10 \
    --config configs/experiments/<name>.yaml \
    --output results/
\end{verbatim}
which persists \texttt{runs.csv}, \texttt{aggregate.json}, and a
\texttt{raw/} subdirectory under \texttt{results/<name>/}. The full
comparison pass is
\begin{verbatim}
chronoagent compare-experiments \
    --output results/ \
    --experiment main_experiment \
    --experiment agentpoison_experiment \
    --experiment baseline_sentinel \
    --ablation ablation_no_bocpd \
    --ablation ablation_no_forecaster \
    --ablation ablation_no_health_scores
\end{verbatim}
which writes the six comparison figures (\texttt{fig1}--\texttt{fig6})
under \texttt{results/figures/} and the two LaTeX tables
(\texttt{table1\_main\_results}, \texttt{table2\_ablation}) under
\texttt{results/tables/}. Those artefacts are \texttt{\textbackslash
input}'d into \cref{sec:results}.
